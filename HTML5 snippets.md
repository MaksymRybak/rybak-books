Struttura base di una pagina web può essere il seguente:
```
<html>
  <body>
    <header>…</header>
    <main>…</main>
    <footer>…</footer>
  </body>
</html>
```

Struttura di una pagina web più completa:
```
<html>
  <body>
    <header>
      <nav>…</nav>
    </header>
    <main>
      <article>
        <section>…</section>
        <aside>…</aside>
      </article>
    </main>
    <footer>
      <nav>…</nav>
    </footer>
  </body>
</html>
```
